```python:chal.py
import os, secrets

P = 2**521 - 1
THRESHOLD = 3
SHARES = 4

flag = int.from_bytes(os.getenv("FLAG", "Alpaca{DUMMY}").encode())
assert flag < P

coeffs = [flag] + [secrets.randbelow(P - 1) + 1 for _ in range(THRESHOLD - 1)]

# f(x) = c0 + c1 * x + c2 * x^2 mod P
f = lambda x: sum(c * pow(x, i, P) for i, c in enumerate(coeffs)) % P

# shares = (1, f(1)), ..., (4, f(4))
shares = [(x, f(x)) for x in range(1, SHARES + 1)]

# 3-out-of-4 secret sharing
print(f"{shares[:THRESHOLD] = }")
```

## Writeup
秘密分散(secret sharing)を用いた問題.

3つの変数$c_{0}$, $c_{1}$, $c_{2}$に対して, $f(1)$, $f(2)$, $f(3)$の値が与えられている.

```Python
coeffs = [flag] + ...
```

の部分からわかるように, $c_{0}$が`flag`の値であるとわかるため, $c_{0}$を求めれば良い.
よって三元一次連立方程式を解く. ここで, `assert flag < P`に注意して, $c_{0}$を$P$で割ったあまりを取る.

$c_{0} = 8329387881342343291890708519937007772772997783746824407965038408368658953123033626348318589$

よってこれを文字列に戻せばよいので, ソルバ例は以下のようになる.

```Python:solve.py
c0 = 8329387881342343291890708519937007772772997783746824407965038408368658953123033626348318589
length = (c0.bit_length() + 7) // 8
flag = c0.to_bytes(length).encode()
print(flag)
```
